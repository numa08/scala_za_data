Playframework 開発者御用達

何がいい？

  • Eclipse と同じ機能や同類のプラグインを入れたときに、比較して早い
  • eclipse/maven/gradle とかのプロジェクトをどれもそのまま取り込める
  • Scala 系プラグインが強力

Scalaプラグインが強力

  • 依存 jar とかから補完とか基本だよね
  • implicit まで見てエラーを出してくれる
    (squeryl なんて DSL すら認識する)
  • implicit まで見て補完対象が出てくる
  • (一度コンパイルすれば)Playframework のテンプレートや router まで補完できる

SBT 対応プラグインがある

  • build.sbt のプロジェクトをそのまま取り込める
  • (完璧ではないけど)build.sbt の設定エラーを検出してくる
  • (完璧ではないけど)build.sbt の記述を補完してくる
  • 実行コンソールを画面内で配置できる

Pac4J

Java 系向け、外部認証ライブラリ群

なにこれ？

Google/Facebook/Twitter とか、色々なサイトからシングルサインオンを簡単に実装するためのライブラリ。
各サイトで認証処理が違ったりするので、個別に対応するのが死ぬほどメンドイ。

HowToUse

例は Playframework のみ

  • Facebook 辺りで開発者登録をしよう(スライドなし)
  • Facebook にアプリを登録しよう(スライドなし)
  • jar を登録する
  • Global.scala で設定
  • router に callback を設定
  • Controller を修正

How To Use(jar を登録する)

説明不要の build.sbt

  "org.pac4j" % "play-pac4j_scala" % "1.2.0-SNAPSHOT",
  "org.pac4j" % "pac4j-oauth" % "1.5.0-SNAPSHOT",

How To Use(Global.scala で設定)

object Global extends GlobalSettings {

  override def onStart(app: Application) {
    Logger.info("Application has started")

    val config = play.api.Play.current.configuration
    val currentHost = config.getString("application.host").getOrElse("http://127.0.0.1:9000")

    val facebookClient = new FacebookClient("ApplicationKey", "ApplicationSecretKey")
    val clients = new Clients(s"$currentHost/callback", facebookClient)

    Config.setClients(clients)
  }

How To Use(router に callback を設定)

GET     /            controllers.Application.index
GET     /login       controllers.Application.login
GET     /callback    org.pac4j.play.CallbackController.callback()
POST    /callback    org.pac4j.play.CallbackController.callback()
GET     /logout      org.pac4j.play.CallbackController.logoutAndRedirect()

How To Use(Controller を修正)

import org.pac4j.play.scala.ScalaController

object Application extends ScalaController {

  def index = Action { implicit request:Request[AnyContent] =>
    val user = Option(getUserProfile(request))
    Ok(views.html.index(user.map(_.getUsername).getOrElse("NoNAME")))
  }

  def login = Action { implicit request:Request[AnyContent] =>

    val session = this.getOrCreateSessionId(request)
    val redirect = getRedirectionUrl(request, session, "FacebookClient", "/")

    Redirect(redirect).withSession(session)
  }
}
